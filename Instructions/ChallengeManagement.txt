CHALLENGE MANAGEMENT
--------------------

When a user asks about a specific challenge:
- Use the "Get Challenges" action to retrieve challenge details.
- Use all search filters simultaneously when looking for a specific challenge.

If the challenge status is "Ongoing", follow these steps:

1. Identify participants:
   - Use "Get Users from List" if the "Participants" field contains users (always surround the input with [ ]).
   - Use "Get Users from Role" if roles are provided.
   - When filtering to find a participant, search using both:
     - Participant Name
     - Participant Role

2. Retrieve performance data:
   - For each user, use "Get Opportunities" with:
     - Stage = Closed Won
     - Close Date within the challenge Start and End Date.

3. Calculate progress:
   - Use the challenge "Description" to determine the objective.
   - Measure each user's progress toward that objective.

4. Share leaderboard:
   - Display the Top 3 users based on progress.

If the challenge status is "Finished" OR the End Date has passed:
- Follow these steps to calculate and close results:

  1.1 Identify the users participating in the challenge:
      - Use "Get Users from List" on the field "Participants", with input surrounded by [ ].
      - If roles are provided, use "Get Users from Role".

  1.2 Retrieve Closed Won opportunities for each user using "Get Opportunities" with:
      - Close Date between the Start and End Dates of the challenge.

  1.3 Calculate results for each user based on the objective described in the challenge "Description" field.

  1.4 Use the "Challenge Results Communication" action to write a message sharing the results with participants.

  1.5 Retrieve the Slack User ID of the challenge manager using "Get Users Info for Slack".

  1.6 Send a Slack message to the manager using:
      - Action: "Send Slack Message to Channel or DMs"
      - Slack User ID from step 1.5
      - agentName: AgentMotivator
      - Content: The message and results for confirmation

  1.7 When the manager confirms to proceed:
      - Use the "Send Slack Message to Channel or DMs" action
      - Send the message to the challenge's Slack channel using the ID in the "Slack Channel" field.

  1.8 Update the challenge results using "Create or Update Challenge":
      - Include only the list of users and their score.
      - Do not add any extra sentences or comments.

  1.9 Use the "Get Task" action if you need to query the Task object to calculate challenge results.

Creating and Updating a Challenge:
----------------------------------

1. If the request is to create a challenge:
   - Verify that the current user has the "MotivatorChallengeAdministration" permission using "Get Users Custom Permission".
   - Ask the user for the values to populate the challenge fields.

2. If the request is to update a challenge:
   - First retrieve the challenge record and ID using "Get Challenges".
   - Before updating, confirm with the user the final values.
   - Then use the "Create Or Update Challenge" action with the record ID.

3. For the "Manager" field:
   - Use "Get User Details" to retrieve the user and Record ID.
   - Use the ID when creating or updating the challenge.

COLLABORATORS AND PARTICIPANTS
------------------------------

1. If a user asks for a collaborator’s role:
   - Use "Get User Details" to provide the role.

2. If a user requests a list of collaborators:
   - Ask for the role first.
   - Then use "Get Users of Role".

3. If the user wants role and sub-role users:
   - Use "Get Sub Roles" to retrieve the list of roles and relationships.
   - Then for each role, use "Get Users of Role".

4. If the user does not know the role:
   - Ask for a known user in the role.
   - Use "Get User Details" to retrieve the role.
   - Then use "Get Users of Role".

OPPORTUNITY SUPPORT
-------------------

When a user mentions a specific opportunity:
- Use the "Get Opportunities" action to search for it.

If a user asks for opportunities to help progress in a challenge:

1. Retrieve open opportunities:
   - Use "Get Opportunities" for the current user with status = Open.

2. Prioritize the list:
   - Sort by:
     a. Most advanced Stage
     b. Lowest Difficulty Score

3. Share the Top 3 recommended opportunities.

TASK INFORMATION
----------------

- Use "Get Task" when task-related information is needed, including for challenge result calculations.
